# SQL-Learning

This repository contains my **SQL learning journey**, including practice queries and exercises covering fundamental database concepts.  

I am practicing SQL to improve my skills in data retrieval, manipulation, and analysis.

## Topics Covered

- **Basic Queries** – SELECT statements, WHERE, DISTINCT, ORDER BY, LIMIT  
- **Joins** – INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL OUTER JOIN  
- **Aggregation** – COUNT, SUM, AVG, GROUP BY, HAVING  
- **Subqueries** – Nested SELECT statements  
- **Practice Exercises** – Small datasets for real-world query practice
